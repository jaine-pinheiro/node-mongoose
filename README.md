## Instalar dependências
```
    npm install
```

## Criar container para o banco MongoDB com Docker
Execute o arquivo para gerar o container:

```
./runDocker.sh
```
*Obs.: O Docker precisa estar instalado. 

## Para iniciar o servidor Express, use o comando:

```
   npm start
```
